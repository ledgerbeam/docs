Connect Ledgerbeam to Figma to export data visualizations and charts directly to your Figma designs, enabling you to create beautiful data-driven designs.

## Features

- Export charts and visualizations to Figma
- Create data-driven design components
- Sync data visualizations automatically
- Share insights through Figma designs

## Setup Instructions

1. Navigate to the Integrations page in your Ledgerbeam dashboard
2. Click on the Figma integration card
3. Click "Connect to Figma" to authorize Ledgerbeam
4. Sign in with your Figma account
5. Grant Ledgerbeam access to your Figma files
6. Click "Allow" to complete the connection

## Usage

Once connected, you can export visualizations to Figma:

1. Run a query in Ledgerbeam that generates a chart or visualization
2. Click the "Export to Figma" option
3. Select the target Figma file and frame
4. Choose the visualization format
5. Your chart will be added to the Figma design

## Supported Visualizations

- Bar charts
- Line charts
- Pie charts
- Tables
- Custom visualizations

## Permissions

The Figma integration requires:
- Read access to view files
- Write access to create and edit frames
- Access to your Figma team files

## Troubleshooting

If you encounter issues:

- Verify your Figma account connection is active
- Check that Ledgerbeam has the necessary file permissions
- Ensure the target Figma file exists and is accessible
- Review the export logs in your Ledgerbeam dashboard
