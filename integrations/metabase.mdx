Connect Ledgerbeam to Metabase to export data insights and analytics directly to your Metabase dashboards and reports.

## Features

- Export query results to Metabase dashboards
- Sync data insights automatically
- Create visualizations from Ledgerbeam queries
- Share analytics with your team through Metabase

## Setup Instructions

1. Navigate to the Integrations page in your Ledgerbeam dashboard
2. Click on the Metabase integration card
3. Enter your Metabase instance URL
4. Provide your Metabase API credentials
5. Click "Connect" to establish the connection
6. Configure which data sources should sync to Metabase

## Usage

Once connected, you can export query results to Metabase:

1. Run a query in Ledgerbeam
2. Click the "Export to Metabase" option
3. Select the target dashboard or create a new one
4. Your data will be available in Metabase

## Configuration

You can configure:
- Which resources should sync to Metabase
- Sync frequency and schedule
- Default dashboard for exports
- Data refresh intervals

## Troubleshooting

If you're experiencing issues:

- Verify your Metabase instance URL is correct
- Check that your API credentials have the necessary permissions
- Ensure your Metabase instance is accessible from the internet
- Review the sync logs in your Ledgerbeam dashboard
