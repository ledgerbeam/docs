Connect Ledgerbeam to Slack to query your data directly from your Slack channels. Simply mention @ledgerbeam in any channel to ask questions about your data and get instant answers.

## Features

- Query data using natural language in Slack channels
- Get instant responses with data insights
- Share results with your team directly in Slack
- Access your data without leaving your workflow

## Setup Instructions

1. Navigate to the Integrations page in your Ledgerbeam dashboard
2. Click on the Slack integration card
3. Click "Connect" to authorize Ledgerbeam to access your Slack workspace
4. Select the channels where you want to enable Ledgerbeam
5. Start querying by mentioning @ledgerbeam in authorized channels

## Usage

Once connected, you can query your data by mentioning @ledgerbeam in any authorized channel:

```
@ledgerbeam What are my top customers this month?
```

Ledgerbeam will process your query and respond with the results directly in the channel.

## Permissions

The Slack integration requires the following permissions:
- Read messages in channels where Ledgerbeam is mentioned
- Post messages in channels where Ledgerbeam is mentioned
- Access basic workspace information

## Troubleshooting

If you're having issues with the Slack integration:

- Ensure Ledgerbeam has been added to the channels where you want to use it
- Check that the integration is properly authorized in your Slack workspace
- Verify your API key is valid and has the necessary permissions
