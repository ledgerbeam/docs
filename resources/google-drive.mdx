Connect your Google Drive to Ledgerbeam to query files and documents using natural language. Google Drive is a file storage and synchronization service.

## Features

- Full Google Drive integration
- Access to files and folders
- Support for various file types
- Secure OAuth authentication

## Connection Requirements

- A Google account with Google Drive
- Network access to Google services

## Setup Instructions

1. Navigate to Resources in your Ledgerbeam dashboard
2. Click "Add Resource" and select "Google Drive"
3. Click "Connect to Google" to authorize Ledgerbeam
4. Sign in with your Google account
5. Grant Ledgerbeam access to Google Drive
6. Select which folders to include (optional)
7. Click "Allow" to complete the connection

## Permissions

Ledgerbeam requires:
- Read access to view files and folders
- Access to file metadata

## Schema Analysis

After connecting, Ledgerbeam automatically:
- Analyzes your Google Drive structure
- Identifies files and their properties
- Understands file types and metadata
- Makes your Drive data queryable

## Querying Drive Data

You can query your Google Drive data using natural language:

- "What files were created this week?"
- "Show me all PDF files"
- "What's the total size of my Drive?"

## Troubleshooting

### Connection Issues

- Verify your Google account connection is active
- Check that Ledgerbeam has the necessary permissions
- Ensure files are accessible
- Verify network connectivity to Google services

### Permission Issues

- Check that Ledgerbeam has Drive read permissions
- Verify file sharing settings
- Ensure files are not restricted

## Best Practices

- Select only the folders you need
- Regularly resync your resource to pick up changes
- Keep your Google account secure
- Monitor Drive access in your Google account
