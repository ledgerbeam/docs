Connect your Airtable bases to Ledgerbeam to query your structured data using natural language. Airtable is a cloud-based database platform.

## Features

- Full Airtable integration
- Access to bases, tables, and records
- Automatic schema detection
- Support for linked records and attachments

## Connection Requirements

- An active Airtable account
- Airtable API key
- Network access to Airtable API

## Setup Instructions

1. Navigate to Resources in your Ledgerbeam dashboard
2. Click "Add Resource" and select "Airtable"
3. Get your Airtable API key:
   - Go to your Airtable account
   - Navigate to Account > Developer hub
   - Create a new personal access token
   - Copy the API key
4. Enter your Airtable API key in Ledgerbeam
5. Select the base you want to connect
6. Click "Test Connection" to verify
7. Click "Create Resource" to complete setup

## Base Selection

After providing your API key, Ledgerbeam will:
- List all accessible bases
- Allow you to select which base to connect
- Analyze the base structure automatically

## Schema Analysis

After connecting, Ledgerbeam automatically:
- Analyzes your Airtable base structure
- Identifies tables and their fields
- Understands field types and relationships
- Makes your Airtable data queryable

## Querying Airtable Data

You can query your Airtable data using natural language:

- "What are the most recent records in the Projects table?"
- "Show me all tasks assigned to John"
- "What's the total value of all opportunities?"

## Troubleshooting

### Connection Issues

- Verify your Airtable API key is correct
- Check that the key has the necessary permissions
- Ensure the base is accessible with your API key
- Verify network connectivity to Airtable API

### Permission Issues

- Check that your API key has access to the base
- Verify base sharing settings
- Ensure you have read permissions

## Best Practices

- Use personal access tokens for individual accounts
- Regularly resync your resource to pick up schema changes
- Monitor API usage in your Airtable account
- Keep your API key secure
