Connect your Intercom workspace to Ledgerbeam to query customer data, conversations, and analytics using natural language. Intercom is a customer messaging platform.

## Features

- Full Intercom API integration
- Access to conversations, users, and companies
- Real-time data access
- Secure API key management

## Connection Requirements

- An active Intercom account
- Intercom API credentials
- Network access to Intercom API

## Setup Instructions

1. Navigate to Resources in your Ledgerbeam dashboard
2. Click "Add Resource" and select "Intercom"
3. Get your Intercom API credentials:
   - Go to your Intercom workspace
   - Navigate to Settings > Developers > Developer hub
   - Create a new app or use existing credentials
   - Copy your Access Token
4. Enter your Intercom Access Token in Ledgerbeam
5. Click "Test Connection" to verify
6. Click "Create Resource" to complete setup

## Available Data

Once connected, you can query:
- Conversations and messages
- Users and contacts
- Companies and accounts
- Teams and admins
- Articles and help center content

## Querying Intercom Data

You can query your Intercom data using natural language:

- "What are the most recent conversations?"
- "Show me users with the highest message count"
- "What's the average response time this week?"

## Security

- API tokens are encrypted before storage
- Connections use HTTPS
- Credentials are never exposed in API responses

## Troubleshooting

### Connection Issues

- Verify your Intercom API token is correct
- Check that the token is active and not expired
- Ensure you have the necessary permissions
- Verify network connectivity to Intercom API

### Permission Issues

- Check that your API token has the required scopes
- Verify your Intercom account is active
- Ensure you have access to the data you're querying

## Best Practices

- Use appropriate API scopes for your needs
- Regularly rotate your API tokens
- Monitor API usage in your Intercom dashboard
- Keep your API credentials secure
